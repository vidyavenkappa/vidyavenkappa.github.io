Projects

Technical Projects

Text Summarisation

Applied machine learning and NLP techniques for abstractive summarization of news articles using models like transformer, encoder-decoder LSTM, and PEGASUS, achieving best results with PEGASUS.

Comprehensive analysis of CSR

Analyzed Corporate Social Responsibility (CSR) data from multiple companies using advanced techniques such as LSA with SVD and TF-IDF. Utilized NLTK for preprocessing and visualized data for performance benchmarking, offering actionable insights.

Sentiment Analysis of Twitter Comments

Extracted sentiment from Twitter comments to gauge public opinions. Cleaned data and used a classification model to categorize comments. Visualized sentiment distribution, revealing public sentiment trends.

Song Popularity Prediction

Predicted song popularity using attributes like acousticness, danceability, etc. Applied various models like Decision Tree, SVM, KNN, XGBoost, and more. The XGBoost Model was especially effective for prediction.

Store Sales Prediction

Predicted sales for an Ecuadorian grocery retailer using historical data and time-series analysis. Used models like ARIMA, Facebook Prophet, and Light Gradient Boosting, with the latter proving most effective.

Cognalyst - Cognitive Assistant

Automated and enhanced user feedback evaluation through the creation of an efficient system that utilized sentiment analysis. Implemented an analytics dashboard to conveniently visualize all the analyzes performed in the reviews, providing a comprehensive overview of the feedback insights.

Thesis

SMACC (A system to streamline multi-modal abstractive and extractive summarization)

The thesis aimed to concisely summarize meetings using LLMs for dialogues up to 90 minutes. I introduced a ’Summary Chunking and Chaining’ technique for managing extensive transcripts and incorporates an extractive feature to generate video trailers that provide users with quick visual context and key takeaways.
